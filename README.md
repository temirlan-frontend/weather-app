# Weather App 🌤️

## 🚀 О проекте

Приложение для просмотра погоды с возможностью поиска города, просмотра почасового и недельного прогноза, а также отображения данных на карте.

Основной фокус — работа с API, обработка данных и построение интерфейса, похожего на реальный продукт.

## 🛠 Стек

- React
- React Router
- Tailwind CSS + Ant Design
- Axios
- Leaflet (карта)

## ⚡ Функциональность

- Поиск города через геокодинг API
- Текущая погода
- Почасовой прогноз
- Прогноз на 7 и 14 дней
- Отображение на карте (Leaflet + OpenStreetMap)
- Сохранение города в localStorage

## 🧠 Что было сложно

- Организовать структуру проекта и разделить логику
